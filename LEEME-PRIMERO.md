## RESUMEN RÁPIDO PARA EL DESARROLLADOR

**Qué es:** Web para empresa de servicios técnicos 24H (electricidad, fontanería, climatización, mantenimiento) en Cataluña. Base en Vilanova i la Geltrú.

**Objetivo de la web:** captar clientes de urgencias y presupuestos. Es una web de CONVERSIÓN (que el visitante llame o pida presupuesto).

**Estilo:** limpio, profesional, azul marino + amarillo. Ver web de referencia en carpeta REFERENCIAS.

**Lo más importante:**
1. RÁPIDA en móvil (la mayoría de urgencias se buscan desde el móvil).
2. Botón de LLAMAR siempre visible + click-to-call.
3. Botón de WhatsApp flotante.
4. Optimizada para SEO local y para conectar con Google Ads (tracking de conversiones).
5. Formulario de contacto corto.
