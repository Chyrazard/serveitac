# TIPOGRAFÍA — ServeiCat 24H

## Fuente del logo
La tipografía del logo es una **sans-serif redondeada y moderna** (tipo Poppins o similar). Usarla como referencia para mantener coherencia con la marca.

## Fuentes recomendadas para la web

### Títulos y encabezados
- **Poppins** (primera opción — coincide con el estilo del logo)
- Alternativas: Montserrat, Raleway
- Peso: SemiBold (600) o Bold (700) para títulos grandes

### Texto de cuerpo
- **Inter** (primera opción — muy legible)
- Alternativas: Open Sans, Roboto
- Peso: Regular (400) para texto, Medium (500) para destacados

## Jerarquía sugerida
| Elemento | Fuente | Tamaño aprox. | Peso |
|----------|--------|---------------|------|
| Título principal (hero) | Poppins | 48-56px | Bold |
| Títulos de sección | Poppins | 32-40px | SemiBold |
| Subtítulos | Poppins | 20-24px | Medium |
| Texto de cuerpo | Inter | 16-18px | Regular |
| Botones | Poppins | 16-18px | SemiBold |

## Notas
- Todas estas fuentes son gratuitas y están en Google Fonts.
- Mantener consistencia: una fuente para títulos (Poppins) y otra para texto (Inter). No mezclar más.
- Priorizar legibilidad en móvil (la mayoría de visitas serán desde el móvil en situaciones de urgencia).
