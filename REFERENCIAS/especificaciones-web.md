## REQUISITOS TÉCNICOS IMPORTANTES

### Optimización móvil (CRÍTICO)
- La web DEBE cargar rápido en móvil — la mayoría de urgencias se buscan desde el móvil.
- Botón de "Llamar" siempre visible sin hacer scroll (above the fold).
- **Click-to-call:** al tocar el teléfono, que llame directamente.

### Conversión
- **Botón de WhatsApp flotante** (esquina, siempre visible).
- CTA de llamada repetidos por toda la página.
- Formulario de contacto CORTO (nombre + teléfono + mensaje). Cada campo extra reduce conversiones.

### SEO local (para que Google encuentre la web)
- Optimizar para búsquedas tipo "fontanero [ciudad]", "fontanero urgente Vilanova", "desatascos Garraf".
- Títulos (H1) que coincidan con lo que busca la gente.
- Preparada para conectar con Google Ads (seguimiento de conversiones / tracking de llamadas).
- Landing pages específicas por servicio si es posible (una para fontanería, una para desatascos, etc.).

### Velocidad
- Imágenes optimizadas (que no pesen y ralenticen).
- Carga rápida = mejor conversión y mejor posición en Google.
