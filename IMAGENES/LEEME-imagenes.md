# IMÁGENES — ServeiCat 24H

## Imágenes a incluir para la web

### Fotos de trabajos reales (lo ideal — cuando las tengas)
- Técnico trabajando en un cuadro eléctrico
- Reparación de fontanería / fuga
- Instalación de aire acondicionado
- Trabajos terminados (antes/después)
- Foto tuya o del equipo con uniforme/logo (da confianza y personaliza)

### Fotos de stock (de momento, hasta tener las reales)
El desarrollador puede conseguir fotos de stock profesionales que encajen con la marca:
- Técnico profesional en cuadro eléctrico (como en la web de referencia)
- Fontanero reparando
- Instalador de climatización
- IMPORTANTE: que las fotos sean coherentes con los colores de marca (tonos azules, profesional, limpio)

## Requisitos técnicos
- Imágenes de buena calidad pero OPTIMIZADAS para web (que no pesen mucho, para que la web cargue rápido).
- Formato JPG o WEBP (mejor WEBP para web).
- Horizontales para el hero, cuadradas o verticales para tarjetas de servicio.

## Nota estratégica
Las fotos de trabajos REALES (con el uniforme y logo de ServeiCat) generan mucha más confianza que las de stock. En cuanto empieces a trabajar, haz fotos de calidad de tus trabajos y del equipo para ir sustituyendo las de stock. También sirven para redes sociales (Instagram) y para el perfil de Google Business.
